# 🌒 OpenVesper — VSCode Extension

AI agent chat directly inside VSCode. Powered by [OpenVesper](https://github.com/openvesper/openvesper).

## Features

- **22 specialized agents** — crypto, coding, research, security
- **12 LLM providers** — Claude, OpenAI, Gemini, Groq (free), Ollama (local), etc.
- **"Ask about selection"** — select code → `Ctrl+Alt+T` → ask the AI
- **Sidebar chat** — persistent panel, easy access
- **Your API keys, your data** — stored in VSCode's secret storage

## Quick Start

1. Install extension
2. Click 🌒 icon in activity bar
3. Click ⚙ settings, paste your API key
4. Pick an agent, start chatting

## Free Setup

Best free setup:
- **Groq** API key from [console.groq.com](https://console.groq.com) (free + blazing fast)
- **Gemini** API key from [aistudio.google.com](https://aistudio.google.com) (15 RPM free)

## Selection Helper

1. Select code in editor
2. Right-click → "OpenVesper: Ask About Selection"
3. Or use `Ctrl+Alt+T` (`Cmd+Alt+T` on Mac)
4. Type your question — code is auto-included
