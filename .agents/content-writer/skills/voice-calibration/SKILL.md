# Voice Calibration

When a user wants content in someone's voice:

## 1. Request samples
At least 2-3 paragraphs of recent writing.

## 2. Analyze across 6 dimensions

| Dimension | What to check |
|-----------|---------------|
| **Sentence length** | Short choppy / long flowing / mixed |
| **Vocabulary** | Casual / professional / technical / colorful |
| **Formality** | "Don't" vs "do not", "I" vs "we" |
| **Quirks** | Em-dashes, parentheticals, ellipses, ALL CAPS, signature words |
| **Energy** | Calm / punchy / urgent / contemplative |
| **Structure** | Hook→story→lesson? Bullet-heavy? Long paragraphs? |

## 3. Build a voice profile

```
VOICE PROFILE — [name]
Avg sentence length: ~15 words, range 6-30
Vocabulary: conversational + occasional technical
Formality: low — uses "you" / "we"
Quirks: em-dashes for asides, opens with questions
Energy: medium-high, declarative
Structure: hook → 3 short points → CTA
Banned words: ["delve", "tapestry", "in conclusion"]
Signature phrases: ["here's the thing", "look —"]
```

## 4. Draft, then audit
After writing:
- Did sentence length match?
- Did formality match?
- Are any banned words present?
- Do any signature phrases appear naturally?

## 5. Show your work
Before delivering, briefly note: "I matched your usual short sentences and em-dash asides. Banned 'delve' and 'tapestry'."

The user trusts you more when they see you noticed their voice.
