# 12-Week Program Design

The discipline: every variable serves a goal.

## 1. Intake (always)

- **Goal:** strength / hypertrophy / fat loss / sport-specific / general fitness
- **Experience:** beginner (<6mo) / intermediate (6mo-2yr) / advanced (2yr+)
- **Sessions per week:** 2 / 3 / 4 / 5 / 6
- **Session duration:** 30 / 45 / 60 / 90 min
- **Equipment:** none / dumbbells / full gym / specialized
- **Injuries:** [list, with current restrictions]
- **Current bests (if any):** squat / bench / deadlift / pull-up / mile time

## 2. Mesocycle structure

A 12-week program typically = 3 mesocycles of 4 weeks each.

```
Weeks 1-4:  Accumulation (volume up, intensity moderate)
Weeks 5-8:  Intensification (volume moderate, intensity up)
Weeks 9-11: Realization (volume down, intensity high)
Week 12:    Deload (50% volume, 70% intensity)
```

## 3. Weekly split (pick based on sessions/week)

### 3-day full body (beginner)
- Mon / Wed / Fri: Squat + Press + Pull pattern each day

### 4-day Upper/Lower
- Mon: Upper (push focus)
- Tue: Lower (squat focus)
- Thu: Upper (pull focus)
- Fri: Lower (deadlift focus)

### 5-day push/pull/legs/upper/lower (intermediate+)
- Mon: Push (chest, shoulders, triceps)
- Tue: Pull (back, biceps, rear delts)
- Wed: Legs (squat + accessories)
- Thu: Upper (mixed)
- Fri: Lower (deadlift + accessories)

### 6-day push/pull/legs ×2 (advanced)
- Push / Pull / Legs / Push / Pull / Legs / Rest

## 4. Exercise selection principles

### Tier 1: Compound lifts (every week, slow progression)
- Squat or front squat
- Deadlift or RDL
- Bench press or incline press
- Overhead press or push press
- Row (barbell, dumbbell, or cable)
- Pull-up or pulldown

### Tier 2: Compound accessories
- Bulgarian split squat
- Romanian deadlift
- Dips
- Single-arm row

### Tier 3: Isolation
- Curls, lateral raises, tricep pushdowns, calf raises, leg curls

Volume guideline per muscle group per week:
- Beginner: 10-12 sets
- Intermediate: 12-18 sets
- Advanced: 16-22 sets

## 5. Progressive overload schedule

For Tier 1 lifts:

| Week | Sets × Reps | %1RM or RPE |
|------|-------------|-------------|
| 1 | 4×6 | 70% / RPE 7 |
| 2 | 4×6 | 72.5% / RPE 7.5 |
| 3 | 4×5 | 75% / RPE 8 |
| 4 | DELOAD 2×8 | 60% / RPE 5 |
| 5 | 4×5 | 75% / RPE 8 |
| 6 | 4×4 | 80% / RPE 8.5 |
| 7 | 3×3 | 85% / RPE 9 |
| 8 | DELOAD 2×8 | 60% / RPE 5 |
| 9 | 3×3 | 85% / RPE 9 |
| 10 | 3×2 | 90% / RPE 9.5 |
| 11 | 2×1+ | 95%+ / RPE 10 (test) |
| 12 | DELOAD | 50% volume |

## 6. Missed sessions protocol

- Missed 1 session: skip, do next session
- Missed 2 in a row: redo most recent missed
- Missed 3+: drop back 1 week in progression
- Sick: take 5-7 days fully off, return at -10% load

## 7. Output Format

Deliver as:
1. **Overview** — goal, structure, expected outcomes
2. **Week-by-week prescription** — sets, reps, intensity
3. **Exercise demos** — link or describe key technique cues
4. **Recovery guidelines** — sleep, protein, deload
5. **Tracking template** — how to log, what to track

Then: "Want me to write Week 1 in full detail, or start with the overview?"
