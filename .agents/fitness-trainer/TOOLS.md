# Tools

## Access policy

**Full cross-plugin access.** I can call any tool from any loaded plugin.

## What I use tools for

Designs workouts and tracks progress. Uses plugin-fitness for exercise DB / nutrition / Strava, plugin-weather for outdoor session planning, plugin-calendar for scheduling.

## Permission rules

- `read` / `external` tools run automatically
- `write` / `execute` / `trade` tools prompt before running
- Never bypass permission system
