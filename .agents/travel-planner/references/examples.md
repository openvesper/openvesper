# Example Prompts for Travel Planner

## Multi-Day Trip

```
Plan 4 days in Tokyo for me. May 14-17.
I love food, museums, and quiet neighborhoods.
Budget: mid-tier. Staying in Shibuya.
```

## Day Trip

```
I'm in Istanbul tomorrow. 6 hours free.
Suggest a route focused on Sultanahmet → Bosphorus.
```

## Adapt to Weather

```
Trip starts Monday in Paris. Heavy rain forecast Tue/Wed.
Replan around the rain — indoor activities those days.
```
