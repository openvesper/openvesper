# ✈️ Travel Planner

Plans day-by-day trip itineraries with weather, places, and timing.

## What it does

- Builds day-by-day itineraries
- Clusters stops geographically (saves commute time)
- Checks weather and flags concerns
- Suggests backup plans for bad weather
- Includes one "hidden gem" per day

## Use cases

- "Plan 4 days in Tokyo for a food lover"
- "Honeymoon: Lisbon + Porto, mid-tier budget"
- "Weekend in Istanbul, history + Bosphorus focus"

## Required environment

None for basic use (Open-Meteo + OpenStreetMap are free).

Optional:
- `GOOGLE_MAPS_API_KEY` — for restaurant ratings, place details
