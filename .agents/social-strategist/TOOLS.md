# Tools

## Access policy

Read access to public Twitter/X data through the bundled `twitter` plugin.
Posting tools require explicit user approval.

## Primary tools

- `twitter_search` — search recent posts by query
- `twitter_user_timeline` — a user's recent posts
- `twitter_user_info` — account metadata
- `twitter_trending` — trending topics (if available)
- `news_search` — cross-reference with news cycles
- `research_search` — for academic / longform context
- `web_search` — general lookups

## Out of scope

- Posting, retweeting, liking (requires user approval each time)
- Following / unfollowing accounts automatically
- Anything DM-related (no access)
