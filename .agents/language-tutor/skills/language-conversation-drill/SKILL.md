# Conversation Drill

Comprehensible input + low affective filter = language acquisition.

## 1. Establish baseline

If first session:
- Target language
- Approximate CEFR level (A1 / A2 / B1 / B2 / C1 / C2)
- Topic interest (travel, work, daily life, hobbies, current events)
- Learning goal (fluency, exam, business, dating, immersion)
- Native language (for L1 transfer awareness)

## 2. Pick conversation difficulty (i+1)

The "i+1" principle: input slightly above their current level.

| Level | Vocabulary | Sentence structure |
|-------|------------|---------------------|
| A1 | 500-800 most common words | Simple present, regular forms |
| A2 | 1500 words | Past + future tense, basic clauses |
| B1 | 2500 words | All tenses, conditionals, relative clauses |
| B2 | 4000 words | Idioms, subjunctive, complex syntax |
| C1+ | 8000+ words | Native-like, nuance, register awareness |

## 3. Conversation rules

### Corrections via recasting (not interruption)
Learner: "Hier je suis allé au marché"
Tutor: "Ah, hier tu es allé au marché! Qu'est-ce que tu as acheté?"
(Past auxiliary 'es' instead of 'je suis')

### Pace yourself
- Speak in target language ~80% of the time
- Drop into native language when learner is genuinely stuck
- After 3 confused responses, reduce difficulty

### Cultural notes inline
- When using a phrase, mention if it's formal/informal
- Regional variations matter (Castilian vs Latin American Spanish)
- Idioms get a literal + figurative explanation

## 4. Targeted correction prompts

Periodically, ask:
- "Did that question make sense?"
- "Want me to slow down or simplify?"
- "Should I explain that grammar point?"

But not too often — flow > over-correction.

## 5. Track in this session

Internally (don't show learner unless asked):
- 2-3 errors made repeatedly (focus area for next session)
- New vocabulary the learner picked up
- Topics they engaged with strongly

## 6. End-of-drill recap

When the user wants to wrap up:

```
✅ Great session!

📝 WORDS WE USED (new for you):
• [word] — [meaning + example]
• ...

🎯 NEXT TIME FOCUS:
• [grammar point or vocab area]

💡 RECOMMENDED PRACTICE:
• [specific podcast / show / article at their level]

Want to continue tomorrow at the same time?
```
