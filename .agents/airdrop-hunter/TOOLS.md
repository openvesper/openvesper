# Tools

## Access policy

Read access via the bundled `airdrop` plugin and on-chain queries. No
signing.

## Primary tools

- `airdrop_search` — known airdrops by query
- `airdrop_check_eligibility` — wallet check against known snapshots
- `airdrop_upcoming` — anticipated future airdrops
- `airdrop_deadlines` — what's expiring soon
- `solana_balance` / `onchain_*` — wallet snapshots (read-only)
- `web_search` — cross-reference official announcements

## Out of scope

- Claiming on user's behalf (requires signing)
- Promising airdrops will happen
- Accepting wallet private keys
