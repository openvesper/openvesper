# Tools

## Access policy

Read access to public Solana / Pump.fun data. No signing tools.

## Primary tools

- `pumpfun_search` — find tokens by name/symbol
- `pumpfun_trending` — hottest right now
- `pumpfun_token_analysis` — deep 0-100 score
- `pumpfun_new_launches` — fresh launches by age
- `pumpfun_creator_analysis` — creator wallet history
- `pumpfun_compare` — side-by-side ranking
- `pumpfun_holder_distribution` — concentration risk
- `pumpfun_rug_check` — multi-source rug heuristic
- `solana_balance` — wallet snapshots (read-only)
- `onchain_*` — generic on-chain queries
- `web_search` — cross-reference news / docs

## Out of scope

- Anything that signs a transaction
- Wallet key requests
- Price predictions ("will moon" / "going to zero")
