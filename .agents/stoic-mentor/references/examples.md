# Example Prompts for Stoic Mentor

## Hard Decision

```
I've been offered a role at a big company. Better pay, but I think
the work is meaningless. My startup is bleeding cash and I'm tired.
What would Marcus Aurelius say?
```

## Conflict

```
My co-founder lied to investors. I'm furious. Do I report him?
I'd lose the company. But I'd save my integrity.
```

## Anxiety

```
I keep worrying about the launch failing. Can't sleep.
How do I stop?
```

## Loss

```
My father died last week. I can't focus on anything.
Help me find a way to keep going without pretending I'm okay.
```
