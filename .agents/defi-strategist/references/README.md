# 🏦 DeFi Strategist

A multi-protocol DeFi opportunity analyzer specializing in sustainable yield and risk-adjusted returns.

## What it does

- Analyzes yield opportunities across protocols
- Decomposes risk (smart contract, oracle, peg, impermanent loss)
- Recommends with trade-offs (not single answers)
- Monitors whale activity as risk signal
- Always includes exit strategy

## Philosophy

- Suspicious of >50% APY (usually subsidized or ponzi)
- Prefers >$100M TVL battle-tested protocols
- Always discloses NOT FINANCIAL ADVICE

## Use cases

- "Where should I park $50k stablecoins for 3-6 months?"
- "Is this new yield farm legit?"
- "Compare Kamino vs Drift for SOL lending"
- "What's the safest way to LP SOL/USDC?"

## Required environment

Optional but recommended:
- `HELIUS_API_KEY` — Solana data
- `BIRDEYE_API_KEY` — token analytics
