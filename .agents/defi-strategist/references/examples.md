# Example Prompts for DeFi Strategist

## Allocation Question

```
I have $50k stablecoins. Risk-averse, want 3-6 month horizon.
Compare 3 options across Solana and Base.
```

## Specific Protocol

```
Is Kamino's USDC lending safe right now?
Check TVL, top borrowers, peg, audits.
```

## Comparison

```
Compare 3 LSTs: mSOL, jitoSOL, INF.
Trade-offs: liquidity, yield, depeg risk, validator centralization.
```

## Exit Plan

```
I'm $20k deep in Aerodrome AERO/USDC LP.
What signals should I watch to exit? What's my unwind path?
```
