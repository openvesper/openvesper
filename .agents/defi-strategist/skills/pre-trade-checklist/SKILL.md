# Pre-Trade Checklist

Before executing ANY trade, run this checklist:

## 1. Token Security
- [ ] `token_security` — GoPlus rug check
- [ ] If Solana: `rugcheck` — RugCheck.xyz score
- [ ] Check honeypot / blacklist functions
- [ ] Verify contract on explorer
- [ ] Liquidity locked / burned?

## 2. Distribution
- [ ] `top_holders` — top 10 hold what %?
- [ ] Dev wallet activity (selling?)
- [ ] Insider concentration (>40% red flag)

## 3. Volume & Liquidity
- [ ] `coin_price` — 24h volume
- [ ] $1k slippage acceptable?
- [ ] Volume on DEX vs CEX

## 4. Sentiment
- [ ] `crypto_twitter_sentiment` — bullish/bearish?
- [ ] `whale_alerts` — accumulation or distribution?
- [ ] `fear_greed_index` — extreme greed = caution

## 5. Technical
- [ ] `technical_analysis` — RSI, MACD, key levels
- [ ] Trend on 1h/4h/1d
- [ ] Recent breakout / breakdown?

## 6. Macro
- [ ] `fear_greed_index` — overall market mood
- [ ] `dxy_yields` — DXY rising = risk-off
- [ ] BTC trend (alt/meme follows)

## 7. Position Sizing
- [ ] Max 2-5% of portfolio for high-risk meme
- [ ] Stop loss defined BEFORE entry
- [ ] Take profit levels (e.g., 25% at 2x, 25% at 5x, runners)

## 8. Mental State
- [ ] Not FOMOing into a 5x candle
- [ ] Not revenge trading
- [ ] Plan written down

If 1-2 boxes fail → wait. If 3+ fail → abort.
