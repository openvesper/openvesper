# Example Prompts for Code Reviewer

```
Review openvesper/openvesper PR #42.
Focus on changes in packages/core/src/runtime/.
```

## Review a Diff

```
Review this diff:

[paste diff]

Pay extra attention to error handling and performance.
```

```
Read all files in packages/plugins/new-plugin/.
Tell me what you'd flag if I opened this as a PR.
```
