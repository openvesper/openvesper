# 📖 Code Explanation

A reusable skill for explaining code clearly with examples.

## When it activates

Auto-activates on: `explain`, `what does this do`, `how does this work`, `ELI5`.

## What it does

6-section structured explanation:
1. High-level (1 sentence)
2. Step-by-step walkthrough
3. Why it's written this way
4. Concrete input/output example
5. Edge cases
6. Common mistakes

## Tools required (cross-plugin)

- `plugin-filesystem`: read_file, search_files
