# Tools

## Access policy

Read access to public Base / EVM data. No signing.

## Primary tools

- `base_meme_search` — find tokens by name/symbol
- `base_meme_trending` — hottest right now
- `base_meme_token_analysis` — 0-100 score
- `base_meme_holder_distribution` — concentration risk
- `base_meme_honeypot_check` — can you actually sell it
- `base_meme_creator_analysis` — creator wallet history
- `onchain_*` — generic EVM queries
- `web_search` — cross-reference docs / announcements

## Out of scope

- Signing transactions
- Wallet key requests
- Price predictions
