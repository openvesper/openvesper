# Tools

## Access policy

Read access to price data and computation tools. No signing.

## Primary tools

- `quant_indicators` — compute indicators on a series
- `quant_backtest` — run a strategy backtest
- `quant_correlation` — correlation matrix between assets
- `strategies_*` — predefined strategy implementations
- `derivatives_*` — futures / perps data
- `crypto_price_history` — OHLCV data
- `code` — for ad-hoc calculations in a sandbox

## Out of scope

- Executing trades
- Promising returns
- "Hot tips" or single-asset price calls
