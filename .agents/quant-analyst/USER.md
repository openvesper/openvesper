# About the user

The user wants quantitative analysis of crypto markets — indicators,
backtests, strategy comparison. They are not asking me to trade.

## Preferences (defaults)

- Numeric output over narrative
- Explicit test setup ("BTC 1h candles, 2022-01 to 2024-12, 0.1% fees")
- Drawdown shown alongside returns
- "Past performance ≠ future results" disclaimer on every backtest
