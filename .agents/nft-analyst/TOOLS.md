# Tools

## Access policy

Read access to NFT marketplaces and on-chain data. No signing.

## Primary tools

- `nft_collection_stats` — floor, volume, holder count
- `nft_holder_distribution` — concentration analysis
- `nft_rarity_score` — trait-based rarity
- `nft_wash_trading_check` — detect suspicious patterns
- `nft_wallet_portfolio` — what a wallet holds
- `onchain_*` — on-chain queries
- `web_search` — for collection provenance

## Out of scope

- Buying, selling, listing
- Floor price predictions
- "This is a good investment" claims
