# Eisenhower Matrix

When user is overwhelmed, sort their tasks into 4 quadrants:

```
              URGENT                 NOT URGENT
            ┌────────────────────┬────────────────────┐
IMPORTANT   │  Q1: DO            │  Q2: SCHEDULE      │
            │  Crises, deadlines │  Strategic work    │
            │  Health emergencies│  Deep work blocks  │
            ├────────────────────┼────────────────────┤
NOT         │  Q3: DELEGATE      │  Q4: DELETE        │
IMPORTANT   │  Interruptions     │  Time wasters      │
            │  Most emails       │  Doom scrolling    │
            └────────────────────┴────────────────────┘
```

## Process

1. List ALL pending tasks (don't filter)
2. For each: is it URGENT? (deadline today/tomorrow)
3. For each: is it IMPORTANT? (moves a key goal forward)
4. Assign to a quadrant

## Allocation

- **Q1**: Do today (but minimize — Q1 = past Q2 procrastination)
- **Q2**: Schedule deep work blocks (most valuable time)
- **Q3**: Delegate or batch (don't drop, just protect time)
- **Q4**: Delete ruthlessly

## Healthy week

- 60% Q2 (strategic)
- 25% Q1 (firefighting)
- 10% Q3 (admin)
- 5% Q4 (recovery, not procrastination)

If Q1 > 40%: you're in chronic crisis mode. Need to invest more in Q2 to prevent future Q1s.
