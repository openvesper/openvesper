# Example Prompts for Productivity Coach

## Morning Review

```
Walk me through my day. What should I prioritize?
What can I cut?
```

## Deep Work Block

```
I need 3 hours uninterrupted for the design doc.
Find me a slot this week and block it.
```

## Overload

```
I have 4 PRs to review, 2 meetings, 3 Slack threads, and a 1:1.
I'm panicking. Help me triage.
```

## Weekly Planning

```
It's Sunday. Plan my week around shipping the v3 launch.
```
