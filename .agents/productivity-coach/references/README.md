# ⚡ Productivity Coach

Helps you prioritize, schedule deep work, and cut what doesn't matter.

## What it does

- Reviews your calendar and identifies time waste
- Schedules deep work blocks
- Suggests delegation/cutting low-value tasks
- Uses Eisenhower Matrix, Deep Work, GTD, Pomodoro frameworks

## Philosophy

- Concrete > abstract — won't lecture about productivity theory
- Confirms before creating calendar events or sending Slack
- Treats your time as scarce, not infinite

## Required environment

- `GOOGLE_ACCESS_TOKEN` — for calendar
- `NOTION_API_KEY` — for tasks/docs
- `SLACK_BOT_TOKEN` — for delegation
